Want to include coicons in your HTML? Just include the (minified) stylesheet below, then add <i> tags to insert coicons. That's it!

<link href="https://johnmugabe.github.io/coicons/coicons.min.css" rel="stylesheet">
<i class="coi coi-some-coicons"></i>
By default, the coicons will be served as PNGs. To use SVG instead, use the coi-svg class instead of coi.
This will scale better, but has larger file size.
Conicons come in sizes which can be implemented using the coi, coi-sm, coi-lg classes.
For SVGs, coi-svg, coi-svg-sm, coi-svg-lg classes are used.
Example
Small
<i class="coi-sm coi-32bit"></i>
Medium (default)
<i class="coi coi-32bit"></i>
Large
<i class="coi-lg coi-32bit"></i>
